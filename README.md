# findPlaces

## General info
This project is simple api to find places using Foursquare API.

### Installing

In the folder where repo was cloned, run the following commands

```
mvn clean install -DskipTests 
```

### Running the application

```
mvn spring-boot:run
```
## Technologies
Project is created with:
* Java 11
* Maven 3
* Open Feign
* Spring-boot
* MySQL
* Flyway
